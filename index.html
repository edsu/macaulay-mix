Macauley Libary Mix

The Macaulay Library is the world's largest and oldest scientific archive of biodiversity audio and video recordings. And it's all on the Web!

The page you are looking at uses HTML5 audio support to provide a constant random stream of the Macauley audio. Build an ecosystem of your own by using the Add Animal button below.

Macauley Library Mix is a simple HTML and JavaScript application, thanks to Macauley Library's permalinks and choice of audio formats. The code is on GitHub if you have ideas for improving the mix.
